FLIXLINK Documentation

WHAT IS FLEXLINK?

FlexLink provides users with information about different movies, directors, and genres. The server-side of the FlixLink application will ensure they have access to this information, that their requests can be processed, and that all necessary data can be stored.

Users will be able to create an account with which to enter, maintain, and update their personal information, as well as create a customized list of their favorite movies.


FLEXLINK API REFERENCE


RETURNING MOVIE DATA

All Movies: GET /movies
Movies by Title: GET /movies/:Title
Genre Info by Genre Name: GET /movies/genre/:genreName:Title
Info About a Director by Name: GET /movies/directors/:directorName

USER ACCOUNTS

Adding a New User: POST /users
Update User Info: PUT /users/:userName
Delete User Account: DELETE /users/:userName

CUSTOMIZING USER FAVORITES

Adding a New Movie to Favorites: POST /users/:userName/movies/:movieID
Removing a Movie from Favorites: DELETE /users/:userName/movies/:movieID
